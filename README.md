# Guess The Number / Rock, Paper, Scissors  
## Overview  
University assignment - This is a simple C++ console application that includes two games:  
1. Guess the Number - the player tries to guess a randomly generated number
2. Rock, Paper, Scissors - the player tries to beat the computer by guessing correctly  
 
This repository is a portfolio version of the original coursework
## Features  
Input validation to prevent errors  
Simple text-based UI  
Simple check system
